# Privacy & Data Handling

**What you'll learn**
- PII handling
- Data minimization

---

## Short summary

Handle PII carefully: redact, minimize retention, and obtain consent before storing user content or keys.

---

## Key concepts

- **Redaction** — Scrub sensitive fields before storing.
- **Retention** — Define TTLs and deletion policies.
- **Consent** — Obtain explicit consent for storage/use.

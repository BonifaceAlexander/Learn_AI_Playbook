# Observability & Monitoring

**What you'll learn**
- Logging & metrics
- Alerting
- Cost monitoring

---

## Short summary

Track metrics (latency, token usage, errors), logs, and set alerts. Use dashboards (Grafana) for visibility.

---

## Key concepts

- **Metrics** — Requests, tokens, latency, errors.
- **Tracing** — Correlate user flows and backend calls.
- **Alerts** — Notify on errors/cost spikes.

---

## Exercise

- Add basic metrics to the backend and visualize with a dashboard.
- Add an alert for high token usage.
